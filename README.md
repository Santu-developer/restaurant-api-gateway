# Restaurant API Gateway

This service acts as the central API Gateway for the Restaurant Digital Ordering System.

It routes incoming client requests to appropriate microservices.

## Responsibilities

- Central entry point
- Request routing
- Load balancing
- Security validation
- Rate limiting

## Technology

- Spring Boot
- Spring Cloud Gateway
- Eureka Service Discovery

## Architecture

Client → API Gateway → Microservices

## Run

Start Config Server  
Start Service Registry  
Start API Gateway

Gateway URL

http://localhost:8080
